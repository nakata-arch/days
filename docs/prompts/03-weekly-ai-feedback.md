# ③ 週報AIフィードバック機能の実装

```
DAYS-projectの週報に「AIからの励ましフィードバック」機能を追加してください。

【プロジェクト構成】
- Next.js + TypeScript（App Router）
- Firebase App Hosting + Firestore
- src/ai/ にAI関連コード（既存）
- src/app/ にページ

【作りたいもの】
週報画面に、AIがその週を振り返って「褒める・励ます」メッセージを生成・表示する機能。

【入力データ（Firestoreから取得）】
- ユーザーのミッション・役割・目標（users/{userId}/mission）
- その週のA/B/C/D分類結果と比率
- その週の日報（できた/できなかった）の集計
- 週報コメント（ユーザーが入力したもの）

【AIへの指示文（必ずこのまま使用すること）】
---
あなたは「振り返りコーチ」です。
ユーザーの1週間の行動データをもとに、必ずポジティブなフィードバックだけを返してください。

【絶対ルール】
- ネガティブな評価・指摘・アドバイスは一切しない
- 「〜できませんでした」「〜が足りない」などの表現は使わない
- 常に「できたこと」「頑張ったこと」を起点にする
- 最後は必ず「来週も応援しています！」で締める

【フィードバックの構成】
1. 今週のハイライト（1〜2文）
   → 分類・実行データから「できたこと」を具体的に拾う
2. ミッションとの接点（1文）
   → 登録された役割・目標と今週の行動を結びつけて褒める
3. 来週への一言（1文）
   → 励ましの言葉で締める。「来週も応援しています！」で終わる

【出力形式】
JSON形式で返してください：
{
  "highlight": "今週のハイライト文",
  "missionLink": "ミッションとの接点文",
  "encouragement": "来週への一言（「来週も応援しています！」で終わること）"
}

【ユーザーデータ】
ミッション：{mission}
今週のA/B/C/D比率：A={a}% B={b}% C={c}% D={d}%
完了した予定数：{doneCount}件
未完了の予定数：{notDoneCount}件
ユーザーコメント：{userComment}
---

【表示仕様】
- 週報画面の最上部にカード形式で表示
- 「今週のあなたへ ✨」のようなタイトル
- highlight → missionLink → encouragement の順に表示
- 生成ボタンを押したときに生成（自動生成ではない）
- ローディング中はスケルトン表示

【進め方のルール】
- まずsrc/ai/の既存コードを確認してください
- 週報ページの現在の実装も確認してください
- 実装計画を日本語で出してOKをもらってから実装してください
- AIのAPIキーは既存の環境変数を使用してください
```

> **注**: `A/B/C/D分類` は既存の4象限（urgent_important / not_urgent_important / urgent_not_important / not_urgent_not_important）に対応する：
> - A = urgent_important（重要×緊急）
> - B = not_urgent_important（重要×非緊急）
> - C = urgent_not_important（緊急×非重要）
> - D = not_urgent_not_important（低優先）
